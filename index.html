Welcome to the Demos section of the PyCav Project.

This repository contains a set of demonstrations written within Python notebooks displaying interactive content from across the NST Physics courses. These could be used within lectures, practicals and supervisions to present the content, conveying the important aspects such as how the behaviour is altered by changing a certain parameter. The notebooks contain code, text, $\LaTeX$ equations, plots and animations hence different perspectives can be emphasised e.g. the equations and figures can be highlighted with little emphasis placed on the code or vice verse.
